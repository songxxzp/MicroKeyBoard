# MicroKeyBoard

An MicroPython Implementation Of [pyKeyboardOS](https://github.com/songxxzp/pyKeyboardOS).
![Key](pictures/poc_2024.png)

## Features
- USB
- BLE

## RoadMap

- Key Mapping
- LED
- LCD Screen
- Full-key rollover HID
- Guided OTA
- WLAN
- MIDI
- SDCARD
